Android中使用ParcelableGenerator存储实体类对象并传输

Android中使用ParcelableGenerator存储实体类对象并传输

标签（空格分隔）： ParcelableGenerator Parcelable 实体类 传输对象

介绍

在安卓中，对象序列化一般有两种方式

  • Serializable，这是在Java中比较常见的一种序列化方式，但是效率比较低
  • Parcelable是Android中提供的，官方推荐的一种方式，效率比Serializable高很多
    虽然Parcelable效率搞，但是使用麻烦，想了解可以参考官方文档或者以下文章
    Android中Parcelable的原理和使用方法
    而ParcelableGenerator可以解决Parcelable使用麻烦的问题

使用方法

  1. 导入依赖
    `dependencies{
    implementation group: 'com.baoyz.pg', name: 'core', version: '2.1.1'
implementation group: 'com.baoyz.pg', name: 'compiler', version: '2.1.1'
    } `
  2. 创建实体类或者已有一个实体类
  3. 实体类对象转化为Parcelable类型，传入intent中
    Intent intent=new Intent(); intent.setClass(ThisActivity.this,NextActivity.class); intent.putExtra("user", PG.convertParcelable(user)); startActivity(intent);
  4. 在跳转到的Activity中
    User user = getIntent().getParcelableExtra("user");
    这就是ParcelableGenerator的使用方式。
